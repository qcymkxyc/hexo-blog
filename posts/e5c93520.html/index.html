EM算法理解

发表于 2022-09-01 | 分类于 数学 |

概述

最近重新看EM算法，看了网上一些博客，发现很多都是抄来抄去，而且我发现其实很多博客上并没有真正讲清楚这个算法的思想，尤其是对于后验分布\(Q(Z|Y)\)(\(Z\)表示隐变量，\(Y\)表示观测变量),并没有解释为什么要这么做，怎么来的，仅仅是照着公式推导了一番而已。这里我按照我的理解写一遍EM算法，或许和网上的其他博客有一些出入。

EM算法全称（Expectation Maximization Algorithm），是一个迭代优化求解算法，在求解概率模型中，只要有隐变量的模型求解几乎都要用到它，例如HMM模型、GMM模型等

问题引入（三硬币模型）

三硬币模型是最简单的含有隐变量的概率模型了，这里还是以三硬币模型举例。

设有3枚硬币，分别记作A、B、C。这些硬币正面出现的概率分别为\(\pi,p,q\).进行如下投硬币试验：先投掷硬币A，根据其结果选出硬币B或硬币C，正面选硬币B，反面选硬币C；然后掷选出的硬币，掷硬币的结果，出现正面记作1，出现反面记作0；独立地重复\(n\)次试验（这里，\(n=10\)）,观测结果如下： \[ 1,1,0,1,0,0,1,0,1,1 \] 求三硬币模型的参数（即求\(\pi,p,q\)）

设观测变量硬币结果为\(Y\)，模型参数为\(\theta\)(即\(\pi,p,q\))，隐变量也就是硬币A投掷的结果为\(Z\)，则有 \[ P(Y;\theta)=\sum_ZP(Y,Z;\theta)=\sum_ZP(Y|Z;\theta)P(Z;\theta) \tag{1} \] 即 \[ P(Y;\theta)=\pi \times p^y \times (1-p)^{1-y}+(1-\pi) \times q^y \times (1-q)^{1-y} \tag {2} \] 对于上式求极大似然函数，有： \[ \mathop{\arg\max_{\theta}}{ \sum_ylogP(Y;\theta)}= \\ \mathop{\arg\max_{\theta}}\sum_ylog[\pi \times p^y \times (1-p)^{1-y}+(1-\pi) \times q^y \times (1-q)^{1-y}] \tag{3} \] 上式由于在\(log\)中有加法，故上式是没有解析解的。

EM算法就是解决上式没有解析解得问题，当然不是用解析解，是用迭代解。

隐变量概率模型的化简

现在还是设观测变量为\(Y\),隐变量为\(Z\),模型参数为\(\theta\),同式（3），求极大似然为 \[ \mathop{\arg\max_{\theta}}{ \sum_ylogP(Y;\theta)} \tag {4} \] 展开为 \[ \mathop{\arg\max_{\theta}}{ \sum_ylog \sum_{Z}P(Y|Z;\theta)P(Z;\theta)} \tag {5} \] 如式（3），上式是没有解析解的。如果上式能够将\(log\)划到对\(Z\)求和的里面，那么此时我们就能够用解析解（实质上最终的EM算法是用的迭代解）。式（5） 利用利用Jensen不等式可以化简为： \[ \mathop{\arg\max_{\theta}}{ \sum_ylog \sum_{Z}P(Y|Z;\theta)P(Z;\theta)} \\ = \mathop{\arg\max_{\theta}}{\sum_y log \sum_Z Q(Z) \frac{P(Y|Z;\theta)P(Z; \theta)} {Q(Z)}} \\ \geqslant \mathop{\arg\max_{\theta}}{\sum_y \sum_Z \left( Q(Z) log{ \left[ \frac{P(Y|Z;\theta)P(Z; \theta)} {Q(Z)} \right] }\right) } \tag{6} \] 其中，\(Q(Z)\)表示关于隐变量\(Z\)的任意分布。

在式（6）中，如果我们能够利用Jensen不等式的相等条件，就能将log函数划入关于\(Z\)求和的里面，此时我们就可以用解析解了。

补充：Jensen不等式

若函数\(f\)是凸函数，对于随机变量\(X\)有下列不等式成立： \[ E[f(X)] \ge f(E[X]) \] 其中，\(E\)表示取期望。

当\(X\)为常数时，不等式的等号成立。

补充：凸函数的定义

凸函数是一个定义在某个向量空间的凸子集\(C\)区间上的实值函数\(f\),如果在其定义域\(C\)上的任意两点\(x_1\),\(x_2\),\(0 \le \lambda \le 1\),有 \[ \lambda f(x_1) + (1 - \lambda)f(x_2) \ge f(\lambda x_1 + (1- \lambda) x_2) \] 则称函数\(f\)为凸函数。

也就是说凸函数的任意两点的割线位于函数图形上方。

Jensen不等式的相等条件为\(log\)函数里面为常数，有 \[ \frac{P(Y|Z;\theta)P(Z; \theta)} {Q(Z)}=c \tag {7} \] 其中，\(c\)表示常数。整理得： \[ Q(Z)=\frac{1}{c}P(Y|Z;\theta)P(Z; \theta) \tag {8} \] 由于\(Q(Z)\)为概率函数，有\(\sum_ZQ(Z)=1\), \[ \sum_ZQ(Z)=\sum_Z\frac{1}{c}P(Y|Z;\theta)P(Z; \theta)=1 \tag{9} \] 上式整理得： \[ c=\sum_ZP(Y|Z; \theta)P(Z;\theta)=P(Y;\theta) \tag {10} \] 联立式（7）和式（10）得： \[ Q(Z)=\frac{P(Y|Z;\theta)P(Z; \theta)}{c}=\frac{P(Y|Z;\theta)P(Z; \theta)}{P(Y; \theta)}=P(Z|Y; \theta) \tag{11} \] 可知，式（11）得到的\(Q(Z)\)是一个关于\(Z\)的后验分布。换句话说，当\(Q(Z)\)为后验分布\(P(Z|Y;\theta)\)时，式（6）中的等号成立。现在把\(Q(Z)\)代入到式（6）中，有 \[ \mathop{\arg\max_{\theta}}{ \sum_ylog \sum_{Z}P(Y|Z;\theta)P(Z;\theta)} \\ = \mathop{\arg\max_{\theta}}{\sum_y \sum_Z \left( P(Z|Y;\theta) log{ \left[ \frac{P(Y|Z;\theta)P(Z; \theta)} {P(Z|Y;\theta)} \right] }\right) } \tag{12} \] 所以，其实引入Jensen不等式的目的是将\(log\)函数划入到关于隐变量\(Z\)的求和里面，实际上，和E步和M步迭代求解的关系并不大。也不是用来确定函数\(\sum_ylog \sum_{Z}P(Y|Z;\theta)P(Z;\theta)\)的下界。

现在我们对式（12）求解析解，由于\(P(Y|Z;\theta)\) 是一个后验分布，在求解时我们需要转化一下，利用贝叶斯公式： \[ P(Z|Y; \theta)= \frac{P(Z,Y; \theta)} {P(Y; \theta)} =\frac{ P(Y|Z; \theta)P(Z;\theta) }{\sum_Z{P(Y|Z; \theta)P(Z; \theta)}} \tag {13} \] 将式（13）代入式（12）中，有 \[ \mathop{\arg\max_{\theta}}{\sum_y \sum_Z \left( \frac{ P(Y|Z; \theta)P(Z;\theta) }{\sum_Z{P(Y|Z; \theta)P(Z; \theta)}} log{ \left[ \frac{P(Y|Z;\theta)P(Z; \theta)} {\frac{ P(Y|Z; \theta)P(Z;\theta) }{\sum_Z{P(Y|Z; \theta)P(Z; \theta)}}} \right] }\right) }= \\ \mathop{\arg\max_{\theta}}{\sum_y \sum_Z \left( \frac{ P(Y|Z; \theta)P(Z;\theta) }{\sum_Z{P(Y|Z; \theta)P(Z; \theta)}} log{ \left[ \frac{P(Y|Z;\theta)P(Z; \theta)\sum_Z{P(Y|Z; \theta)P(Z; \theta)}} { P(Y|Z; \theta)P(Z;\theta) } \right] }\right) } \tag {14} \] 实质上，将参数代入式（14）相当之复杂，如果利用拉格朗日对上式在求一个偏导数，就更加复杂了。你可以将上面的三硬币模型的参数代入其中，尝试一下。对于像GMM ,HMM模型的参数，就更复杂了，所以需要引入迭代解。

其实EM算法引入迭代解和Jensen不等式的关系并不大，Jensen不等式也不是用来确定似然函数的下界的，最多在收敛性的证明部分用得到

EM迭代的引入

令式（12）为函数\(G(\theta)\)： \[ \mathop{\arg\max_{\theta}}{\sum_y \sum_Z \left( P(Z|Y;\theta) log{ \left[ \frac{P(Y|Z;\theta)P(Z; \theta)} {P(Z|Y;\theta)} \right] }\right) }=\mathop{\arg\max_{\theta}}{G(\theta)} \tag{15} \] 要使迭代成立，我们仅需第\(i+1\)次迭代的结果大于第\(i\)次迭代的结果即可，有 \[ G(\theta^{(i+1)}) \ge G(\theta^{(i)}) \tag {16} \] 换句话说，我们仅需要找到一种迭代方式，使得式（16）成立即可，用更通俗的话说，就是保证函数\(G(\theta)\)每一步都在爬楼梯。在EM算法中，先用第\(i\)次的迭代计算\(P(Z|Y; \theta)\),

也就是： \[ P(Z|Y; \theta^{(i)})=\frac{P(Z,Y; \theta^{(i)})} {P(Y; \theta^{(i)})} =\frac{ P(Y|Z; \theta^{(i)})P(Z;\theta^{(i)}) }{\sum_Z{P(Y|Z; \theta^{(i)})P(Z; \theta^{(i)})}} \tag {17} \] 由于\(\theta^{(i)}\)我们是知道的，所以我们可以计算出\(P(Z|Y; \theta^{(i)})\),这就是EM算法的E步。

计算出\(P(Z|Y; \theta^{(i)})\)后，\(P(Z|Y; \theta^{(i)})\)就可以看成常数了，此时求\(G(\theta)\)的极大似然函数就较为简单： \[ \mathop{\arg\max_{\theta}}{G(\theta)}=\mathop{\arg\max_{\theta}}{\sum_y \sum_Z \left( P(Z|Y;\theta^{(i)}) log{ \left[ \frac{P(Y|Z;\theta)P(Z; \theta)} {P(Z|Y;\theta^{(i)})} \right] }\right) } \tag {18} \] 这就是EM算法的M步。

综上，

E步为计算第\(i\)轮的后验分布： \[ P(Z|Y; \theta^{(i)})=\frac{P(Z,Y; \theta^{(i)})} {P(Y; \theta^{(i)})} =\frac{ P(Y|Z; \theta^{(i)})P(Z;\theta^{(i)}) }{\sum_Z{P(Y|Z; \theta^{(i)})P(Z; \theta^{(i)})}} \] M步为计算加入后验估计后的极大似然： \[ \mathop{\arg\max_{\theta}}{\sum_y \sum_Z \left( P(Z|Y;\theta^{(i)}) log{ \left[ \frac{P(Y|Z;\theta)P(Z; \theta)} {P(Z|Y;\theta^{(i)})} \right] }\right) } \]

收敛性证明

收敛性证明即证明式（16）成立.

最初的我们要求极大似然函数为： \[ L(\theta)= \sum_ylog \sum_{Z}P(Y|Z;\theta)P(Z;\theta) \tag {19} \] 第\((i +1)\)轮Jensen不等式的相等条件为： \[ L(\theta^{(i+1)})= \sum_ylog \sum_{Z}P(Y|Z;\theta^{(i+1)})P(Z;\theta^{(i+1)})=\\ ={\sum_y \sum_Z \left( P(Z|Y;\theta^{(i+1)}) log{ \left[ \frac{P(Y|Z;\theta^{(i+1)})P(Z; \theta^{(i+1)})} {P(Z|Y;\theta^{(i+1)})} \right] }\right) } = G(\theta^{(i+1)}) \\ ={\sum_y \sum_Z \left( Q(Z;\theta^{(i+1)}) log{ \left[ \frac{P(Y|Z;\theta^{(i+1)})P(Z; \theta^{(i+1)})} {Q(Z;\theta^{(i+1)})} \right] }\right) } \tag {20} \] 对于式（20），由Jensen不等式，当\(Q(Z;\theta)\)不取\(\theta^{(i+1)}\)时，必然有其值小于\(L(\theta^{(i+1)})\),故有 \[ L(\theta^{(i+1)}) > {\sum_y \sum_Z \left( Q(Z;\theta^{(i)}) log{ \left[ \frac{P(Y|Z;\theta^{(i+1)})P(Z; \theta^{(i+1)})} {Q(Z;\theta^{(i)})} \right] }\right) } \tag {21} \] 由于\(\theta^{(i+1)}\)来源于在给定了\(\theta^{(i)}\)之后估计的最大值（即M步，见式（18）），所以一定有： \[ {\sum_y \sum_Z \left( P(Z|Y;\theta^{(i)}) log{ \left[ \frac{P(Y|Z;\theta^{(i+1)})P(Z; \theta^{(i+1)})} {P(Z|Y;\theta^{(i)})} \right] }\right) } \ge \\ {\sum_y \sum_Z \left( P(Z|Y;\theta^{(i)}) log{ \left[ \frac{P(Y|Z;\theta^{(i)})P(Z; \theta^{(i)})} {P(Z|Y;\theta^{(i)})} \right] }\right) } = G(\theta^{(i)})\tag {22} \]

故有： \[ G(\theta^{(i+1)}) \ge G(\theta^{(i)}) \] 故式（16）得证。

对于《统计学习方法》的疑惑

李航博士的《统计学习方法》（第二版）对于EM算法的讲解和网上以及CS229上的略有不同，

书中主要是利用\(L(\theta) - L(\theta^{(i)})\)的方式来衡量当前迭代次数\(i\)和最终结果的差异\(s\)，然后利用差异\(s\)反过来表示\(L(\theta)\),再求最大值。（而在CS229中其实是直接求\(\mathop{\arg\max_{\theta}}{L(\theta)}\)）.即 \[ L(\theta) = L(\theta^{(i)}) + s \tag {23} \]

\[ \mathop{\arg\max_{\theta}}{L(\theta)} =\mathop{\arg\max_{\theta}}{[L(\theta^{(i)}) + s]} \tag {24} \]

但在 179页，利用Jensen不等式得到下界，为何凑的任意分布就一定是\(P(Z|Y,\theta^{(i)})\)（虽然用上述的方式得到的结果和这里相同） \[ L(\theta) - L(\theta^{(i)})= log \left( \sum_ZP(Y|Z,\theta)P(Z|\theta) \right) - logP(Y| \theta^{(i)}) \tag {25} \] 上式利用Jensen不等式得到下界 \[ L(\theta) - L(\theta^{(i)}) = log \left( \sum_ZP(Z|Y,\theta^{(i)}) \frac{P(Y|Z,\theta)P(Z| \theta)}{P(Z|Y,\theta^{(i)})} \right) - logP(Y| \theta^{(i)}) \\ \ge \sum_ZP(Z|Y, \theta^{(i)})log\frac{P(Y|Z,\theta)P(Z| \theta)}{P(Z|Y,\theta^{(i)})}-logP(Y| \theta^{(i)}) \tag {26} \] 式（25）和（26）中，\(Z\)表示隐变量，\(Y\)表示观测变量，\(\theta^{(i)}\)表示第\(i\)次迭代的参数。式（26）为何凑的分布就一定是\(P(Z|Y,\theta^{(i)})\).
