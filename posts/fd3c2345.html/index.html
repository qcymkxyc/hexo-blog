脉证十六讲总结

发表于 2023-01-12 | 分类于 中医 |

摸脉顺序：

  1. 全候浮取 -> 全候中取-> 全候沉取（调整中取力度与位置）

  2. 全候中取 -> 全候浮取 ->全候中取 -> 全候沉取（判断总体脉象）

  3. 寸脉，浮中沉

  4. 关脉，浮中沉

  5. 尺脉，浮中沉

正常脉象及影响因素

  • 女性脉较男性细弱，且至数稍快，脉形较细小
  • 越小，脉越快，三岁以内的小孩子，脉跳每分钟一百次左右正常；五六岁，每分钟八九十次正常，青年人脉象较大而有力，老年人多微弦
  • 身材高大者，脉现较长，矮小者，脉现较短，廋人多偏浮，胖人多偏沉，运动员，脉多缓而有力，甚至迟脉
  • 喜则气缓而脉多缓；怒则气上而脉多弦，惊则气乱而脉动无序
  • 酒后，脉象变数，或滑，或滑数，甚至脉稍大
  • 四季脉象微有倾向性，分别为弦洪浮沉
  • 尤其是降压药之类药，对脉象影响极大

实脉

定义：脉搏有力程度超过正常人而不够软者

邪气盛实，正气不虚

实证分为表里，表证以外感居多

  1. 里实证，舌红苔黄，有明显的热证，如火热之实证
  2. 舌苔不黄，舌质不红，为有形之物阻滞，痰饮，气滞，食滞均有可能

邪气虽盛，正气大亏

  • 伤寒，下利日十余行，脉反实者，死
  • 反强之象（此时通过望诊，可看到病人极虚）脉证不符，当特别注意

弹指脉

阳盛躁动，化生火热

形成机理：“阳气者，劳烦则张”，阳气躁动，则有亢无制，必化火，故与数脉同见

  • 一旦化火，就有内脏出血的风险
  • 一部分高血压病人就见弹指脉

虚气浮动

主要指脾气和心气的浮动

形成机理：虚气浮动，气为血之帅，气无所敛，浮越于外

  • 弹指脉多兼虚大之象，或直接表现为芤脉
  • 也可能脉大弹指而有力，需四诊合参

风痰鼓动

形成机理：

  • 常兼有：弦脉，滑脉，股指脉，曲脉
  • 兼弦脉，因为弦主风
  • 兼滑脉，滑主痰
  • 兼曲脉，曲主风痰鼓舞
  • 多数不兼数脉，也可以兼数脉，但情况少见

但姚梅龄先生并没有讲为什么多数不兼数脉，既然风痰鼓动可以兼热兼火，应该有数脉才对。或许这是临床经验所得

虚脉

定义：浮取较为明显，但较软，较之正常脉和软脉更无力，且中取沉取明显无力，甚至沉取不应指

气虚

血脱

形成机理：气随血脱，本质上包含气虚

此时多见芤脉

阳虚

形成机理：阳气皆虚

阴将枯竭

气阴两虚

多兼见涩脉，芤脉

津虚

多见气津两虚

燥热后津大亏

汗出过多，阴阳两虚

精亏

精在阳气的作用下而生气

弱脉

定义：浮取指下感觉不明显，中取沉取虽然能感觉到明显的搏动，但均无力

和虚脉的区别：沉以候弱，浮以候虚

脾胃虚弱

阳虚

脏腑之气虚

阴液虚

阴液虚可能出现弱脉

邪退正复

《伤寒论》第358条：“下利，有微弱而渴，脉弱者，令自愈”

《伤寒论》第363条：”下利脉沉弦者，下重也。脉大者为未止，脉微弱数者为欲自止“

形成机理：正邪相争，正气胜，但正气大伤，故出现弱脉

微脉

定义：脉象极其微弱，按之欲绝

阳气衰微，但还没有到阳气欲绝的地步

《伤寒论》258条：“少阴之为病，脉细微，但欲寐也”

持续几个小时到12个小时左右

少阴伤寒，有死症

气随血脱，元阳暴脱，气阴亡脱

极其危险，一般仅持续数分钟到一个小时左右

脉率：

  1. 疾脉 7至以上（成人）
  2. 数脉 5至以上
  3. 缓脉 4至
  4. 尺脉 4至以下

脉律：

  1. 代脉，跳几下停一下，但止有定数，脉律是均匀的

  2. 促脉， 数中一止

  3. 结脉，迟中一止

  4. 促脉和结脉止无定数

  5. 三五不调脉，脉搏不均，（三五不调，时时歇止）

    气血虚夹有风，或夹有淤，或夹有痰饮

  6. 咋数咋疏脉

脉位：

  1. 浮
  2. 中
  3. 沉
  4. 伏

脉体：

​ 大，细，长，短，曲，双

脉力：

  • 饮食集聚，见右关独浮弦旺有力

大脉

  • 邪盛病进大脉，必大而有力
  • 正气大亏者，兼有虚象（脉象中空而不受按）
  • 虚阳外越者，脉可浮大而空

短脉

  • 寸短或尺短，或寸尺都短（不满指）
